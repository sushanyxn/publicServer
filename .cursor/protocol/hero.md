# 英雄 协议

> 协议号段：1400-1499
> 来源包：`com.slg.net.message.clientmessage.hero.packet`

## 协议列表

### HeroVO

- **协议号**：1400
- **类型**：数据体(VO)
- **说明**：英雄 VO
- **字段**：
  - `heroId` (int) — 英雄配置 id
  - `heroLv` (int) — 英雄等级

### CM_HeroLevelUp

- **协议号**：1401
- **类型**：客户端请求
- **说明**：英雄升级请求，客户端发送给服务端，请求对指定英雄进行升级
- **字段**：
  - `heroId` (int) — 要升级的英雄配置 id

### SM_HeroInfo

- **协议号**：1402
- **类型**：服务端推送
- **说明**：英雄信息全量推送，登录时服务端推送给客户端的全部英雄数据
- **字段**：
  - `heroes` (HeroVO[]) — 玩家拥有的所有英雄

### SM_HeroUpdate

- **协议号**：1403
- **类型**：服务端推送
- **说明**：英雄变更增量推送，英雄获得或升级时服务端推送给客户端的单个英雄数据
- **字段**：
  - `hero` (HeroVO) — 变更的英雄数据
