# 登录 协议

> 协议号段：1000-1099
> 来源包：`com.slg.net.message.clientmessage.login.packet`

## 协议列表

### CM_LoginReq

- **协议号**：1001
- **类型**：客户端请求
- **说明**：登录请求消息，客户端发送给服务端的登录请求
- **字段**：
  - `account` (String) — 账号（登录标识）
  - `playerId` (long) — 玩家 id（选服/重连时可选填，0 表示新登录）
  - `loginToken` (String) — 登录令牌

### SM_LoginResp

- **协议号**：1002
- **类型**：服务端推送
- **说明**：登录响应消息，服务端发送给客户端的登录响应
- **字段**：
  - `code` (int) — 响应码（0 表示成功，非 0 表示失败）
  - `playerId` (long) — 登录成功时的玩家 id，失败时为 0

### CM_LoginFinish

- **协议号**：1003
- **类型**：客户端请求
- **说明**：登录完成消息，客户端在收到 SM_LoginResp 成功后发送，表示正式完成登录；服务端在此后抛出登录事件
- **字段**：无
