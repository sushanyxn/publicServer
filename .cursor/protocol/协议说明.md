# 协议说明

本文档描述 slg-net 模块下所有 packet 协议的作用，协议号与类名以 `message.yml` 为准。

---

## 索引

| 模块 | 协议数量 | 协议号范围 |
|------|----------|------------|
| [rpc](#rpc-模块-rpc) | 2 | 101-102 |
| [socket](#socket-模块-socket) | 2 | 111-112 |
| [login](#登录模块-login) | 2 | 1001-1002 |
| [scene](#场景模块-scene) | 21 | 1100-1120 |

---

## RPC 模块 (rpc)

服务器内部 RPC 通信消息。

| 协议号 | 类名 | 类型 | 作用 |
|--------|------|------|------|
| 101 | IM_RpcRequest | 内部请求 | RPC 请求消息，支持 Deadline 机制（借鉴 gRPC），携带来源服务器 ID、方法标识、参数等 |
| 102 | IM_RpcRespone | 内部响应 | RPC 响应消息，携带请求 ID、响应结果及异常信息 |

---

## SOCKET 模块 (socket)

内部链接会话注册相关消息。

| 协议号 | 类名 | 类型 | 作用 |
|--------|------|------|------|
| 111 | IM_RegisterSessionRequest | 内部请求 | 内部链接注册请求，携带来源服务器 ID |
| 112 | IM_RegisterSessionResponce | 内部响应 | 内部链接注册返回，携带注册结果 |

---

## 登录模块 (login)

客户端登录流程相关协议。

| 协议号 | 类名 | 类型 | 作用 |
|--------|------|------|------|
| 1001 | CM_LoginReq | 客户端请求 | 登录请求，客户端发送给服务端，携带账号、玩家 ID 等信息 |
| 1002 | SM_LoginResp | 服务端推送 | 登录响应，服务端返回登录结果（响应码 0 表示成功）、玩家 ID |

---

## 场景模块 (scene)

场景进入、视野、节点与各类 VO 数据。

| 协议号 | 类名 | 类型 | 作用 |
|--------|------|------|------|
| 1100 | AllianceOwnerVO | 数据体(VO) | 联盟所有者 VO，继承 OwnerVO，包含联盟 ID、短名、名称、旗帜等 |
| 1101 | ArmyVO | 数据体(VO) | 军队 VO 抽象基类，包含军队 ID |
| 1102 | AssembleArmyVO | 数据体(VO) | 集结军队 VO，继承 ArmyVO，包含队长 ID、成员列表（PlayerArmyVO） |
| 1103 | CM_Watch | 客户端请求 | 场景查看协议，携带视野移动后的坐标（x、y）及层级等 |
| 1104 | MonsterOwnerVO | 数据体(VO) | 怪物所有者 VO，继承 OwnerVO，包含配置 ID |
| 1105 | OwnerVO | 数据体(VO) | 所有者 VO 抽象基类 |
| 1106 | PlayerArmyVO | 数据体(VO) | 玩家军队 VO，继承 ArmyVO |
| 1107 | PlayerCityVO | 数据体(VO) | 玩家城市 VO，继承 StaticNodeVO |
| 1108 | PlayerOwnerVO | 数据体(VO) | 玩家所有者 VO，继承 OwnerVO，包含玩家 ID、联盟 ID、头像 ID、城市位置等 |
| 1109 | PositionVO | 数据体(VO) | 位置 VO，包含 x、y 坐标 |
| 1110 | ResourceNodeVO | 数据体(VO) | 资源节点 VO，继承 StaticNodeVO |
| 1111 | RouteNodeVO | 数据体(VO) | 路径节点 VO，继承 SceneNodeVO，包含起点、终点、军队信息 |
| 1112 | SceneNodeVO | 数据体(VO) | 场景节点 VO 抽象基类，包含节点 ID、所有者 |
| 1113 | SM_SceneNodeAppear | 服务端推送 | 场景节点出现消息，服务端推送新出现的场景节点（SceneNodeVO） |
| 1114 | SM_SceneNodeDisappear | 服务端推送 | 场景节点消失消息，服务端推送消失的节点 ID |
| 1115 | StaticNodeVO | 数据体(VO) | 静态节点 VO 抽象基类，继承 SceneNodeVO，包含位置 |
| 1116 | WorldBossArmyVO | 数据体(VO) | 世界 Boss 军队 VO，继承 ArmyVO，包含配置 ID |
| 1117 | ScenePlayerVO | 数据体(VO) | 需透传到场景的玩家信息，包含玩家 ID、游戏服 ID |
| 1118 | CM_LoadSceneFinish | 客户端请求 | 客户端加载地图完成，正式进入场景的通知 |
| 1119 | SM_EnterScene | 服务端推送 | 请求进入场景的返回，result=0 表示校验通过可切图，其他为错误码 |
| 1120 | CM_EnterScene | 客户端请求 | 请求进入场景，携带服务器 ID、场景 ID |

## 任务模块 (task)

任务领取与进度推送。

| 协议号 | 类名 | 类型 | 作用 |
|--------|------|------|------|
| 1200 | CM_GainTask | 客户端请求 | 领取任务奖励请求，客户端完成任务后请求领取奖励，携带任务类型、任务 ID |
| 1201 | SM_UpdateTask | 服务端推送 | 任务更新推送，服务端推送任务进度更新（TaskVO） |
| 1202 | SM_MainTaskInfo | 服务端推送 | 主线任务信息推送，服务端推送主线任务列表 |
| 1203 | TaskVO | 数据体(VO) | 任务值对象，客户端显示任务信息的数据传输对象，包含任务 ID、类型、进度等 |

---

*文档由协议说明流程根据 slg-net 下 packet 类与 message.yml 自动生成，仅作说明用途。*
