# 测试结果目录

本目录存放各次**框架测试的执行结果**，供回归对比与框架测试 Skill 参考。

## 命名建议

- 按框架 + 日期：`{框架}-result-{yyyyMMdd}.md` 或 `{框架}-result-{yyyyMMdd-HHmm}.json`
- 或按流水：`result-{yyyyMMdd-HHmmss}.md`，内容中区分各框架

## 结果内容建议

- 执行时间、涉及模块、环境（单元/集成）
- 用例总数、通过数、失败数、跳过数
- 失败用例名称与原因（或链接到 CI 日志）
- 与上次结果的差异（新增/修复/仍失败）

框架测试 Skill 在执行新一轮测试后会在此更新或追加结果文件。
